About

Hey, you are seeing a demo of Responsive Tabs jQuery plugin (developed by Anup Pandey).

This plugin makes any markup of yours into Responsive Tabs. On mobile, the tabs are converted into dropdown!

To know how to use this plugin, see "How to use" tab!

How to use

To use this plugin, simply:
  1. Call method responsiveTabs() from your script to the markup you want to make as Responsive Tabs
    For example: $('#respTabs').responsiveTabs() in this case

  2. You can set one of the tabs as the default tab you want to see as the active tab when you land on the page.
    To know how, see "Default" tab

Default

Hello, I've been set as default by adding the attribute resp-tab = "default" in my parent div
